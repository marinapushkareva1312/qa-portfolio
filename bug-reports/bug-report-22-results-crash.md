# Bug Report #22 — Page Freezes When Clicking Results Number Buttons

Date: 2026-05-31  
Tester: Marina Pushkareva  
Website: <https://academybugs.com>

-----

## Summary

The page becomes unresponsive when clicking the buttons to change the number of displayed results.

-----

## Bug Details

|Field          |Value       |
|---------------|------------|
|**Severity**   |High        |
|**Issue Type** |Crash       |
|**Frequency**  |Every Time  |
|**Environment**|All Browsers|

-----

## Steps to Reproduce

1. Open <https://academybugs.com>
1. Click the Find Bugs link on the navigation bar
1. Click on the buttons at the top left to show a certain number of results

-----

## Expected Result

The selected number of results is displayed according to the clicked buttons.

-----

## Actual Result

The page freezes when clicking on the numbers of results buttons. The user cannot interact with the page after this action.
