# Bug Report #21 — Page Freezes When Changing Currency

Date: 2026-05-31  
Tester: Marina Pushkareva  
Website: <https://academybugs.com>

-----

## Summary

The page freezes when the user attempts to change the currency in the right side menu on a product page.

-----

## Bug Details

|Field          |Value       |
|---------------|------------|
|**Severity**   |High        |
|**Issue Type** |Crash       |
|**Frequency**  |Every Time  |
|**Environment**|All Browsers|

-----

## Steps to Reproduce

1. Open <https://academybugs.com>
1. Click the Find Bugs link on the navigation bar
1. Open any product page
1. Locate the currency selector in the right side menu
1. Change the currency

-----

## Expected Result

The currency is changed as expected and the page continues to function normally.

-----

## Actual Result

The page freezes when changing the currency. The user cannot interact with the page after this action.
