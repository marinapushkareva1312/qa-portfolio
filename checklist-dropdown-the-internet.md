## Checklist

### Visibility & Display

- [ ] Dropdown element is visible on the page
- [ ] Dropdown has a default selected option (“Please select an option”)
- [ ] All options are visible when dropdown is opened
- [ ] Dropdown label/title is present and readable

### Functionality

- [ ] Dropdown opens on click
- [ ] Option 1 can be selected
- [ ] Option 2 can be selected
- [ ] After selecting an option, it appears as the selected value
- [ ] Dropdown closes after selecting an option
- [ ] Dropdown closes when clicking outside of it

### Keyboard Navigation

- [ ] Dropdown can be opened with keyboard (Tab + Enter/Space)
- [ ] Options can be navigated with arrow keys (↑ ↓)
- [ ] Option can be selected with Enter key
- [ ] Dropdown can be closed with Escape key

### UI & Design

- [ ] Dropdown is aligned correctly on the page
- [ ] Dropdown styling is consistent with the page design
- [ ] Selected option is clearly highlighted
- [ ] Dropdown does not overlap other page elements when open

### Edge Cases

- [ ] Selecting the same option twice does not cause any errors
- [ ] Page does not reload or crash after selecting an option
